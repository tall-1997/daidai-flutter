## 呆呆面板 Flutter App v0.0.38

### 更新内容
新增功能模块：
- **仪表盘**：概览统计、快捷操作、最近执行任务、系统资源监控
- **订阅管理**：Git 仓库订阅、定时同步、SSH Key 认证、白名单/黑名单过滤
- **个人设置**：修改密码、修改用户名、双因素认证 (2FA)、会话管理
- **用户管理**：用户列表、创建/编辑/删除用户、角色管理（管理员/普通用户）
- **开放 API**：API Key 管理、App Key/Secret 查看、使用说明

功能完善：
- 更新导航结构，仪表盘作为首屏
- 添加订阅、开放API、用户、我的等导航项
- 完善深色模式适配

### 下载说明
- **Android**: 下载 APK 文件安装使用
- **iOS**: 下载 IPA 文件，通过 AltStore 或 Sideloadly 安装
