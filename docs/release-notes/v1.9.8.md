# v1.9.8

## 更新内容

- 依赖管理支持多选重装，并按顺序逐个执行，避免并发重装导致再次失败。
- 依赖管理新增导出能力，可直接导出已安装依赖清单，格式为 `依赖名==>版本号`。
- 环境变量页面补充前端样式优化，缓解特定缩放比例下的卡顿问题。
- 会话管理增强客户端识别，区分 Web / App，并尽量显示更具体的设备型号与客户端信息。
- 脚本通知 helper 改为统一复用脚本根目录下的 `notify.py` / `sendNotify.js`，不再在每个脚本目录重复创建副本。
- 脚本管理的历史版本新增“对比”按钮，可左右查看历史代码与当前代码差异，并用红绿高亮展示改动。
- 调试运行与代码运行器在关闭弹窗、离开页面、刷新或关闭浏览器时，会主动停止后端运行进程，避免后台残留执行。
