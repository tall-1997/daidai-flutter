## 面板优化

### 定时任务界面优化

- 禁用按钮改为红色（`danger` 类型），状态更加醒目
- 所有操作按钮 padding 加大，提升点击舒适度

### Open API 新增订阅管理权限

- 权限范围新增「订阅管理（subscriptions）」选项
- 后端订阅路由添加 `OpenAPIAccess("subscriptions")` 中间件

### 脚本管理拖拽增强

- 脚本文件现在可以从文件夹直接拖拽回根目录（之前只能拖入文件夹）
- 拖拽到根级节点的前后位置时自动移动到根目录
- 恢复拖放指示线显示

### 系统设置标签页重命名

重新命名各标签页，语义更清晰：

| 原名称 | 新名称 |
|--------|--------|
| 系统配置 | 面板外观 |
| 任务执行 | 任务运行 |
| 网络代理 | 代理设置 |
| 验证码设置 | 登录验证码 |
| 数据备份 | 备份恢复 |
| 安全 | 账号安全 |

### Monaco 编辑器右键菜单中文化

通过 `vs/nls` 配置将 Monaco Editor 的 locale 设置为 `zh-cn`，右键上下文菜单（剪切、复制、粘贴、命令面板等）全部显示中文。

### 日志查看器滚动位置保持

修复定时任务日志窗口在标签页切换后自动回到顶部的问题：

- SSE 重连时保存并恢复 `scrollTop` 位置
- 新增 `pendingScrollRestore` 机制，在首批数据刷新后恢复滚动位置
- 仅在非重连（首次打开）时才滚动到顶部

### 加密脚本执行支持优化

针对 safekodo 等加密工具生成的超长单行脚本：

- Scanner 缓冲区从 1MB 扩大到 16MB
- 单行超过 8KB 时截断显示，避免日志被无意义的加密代码填满
- Scanner 超限后降级读取剩余输出（最多 32KB），确保 Node.js 的错误信息不丢失

## Magisk 模块

### SSH 用户名密码配置

`/data/adb/daidai-panel/ports.conf` 新增两个配置项：

| 变量 | 默认值 | 说明 |
|------|--------|------|
| SSH_USER | root | SSH 登录用户名 |
| SSH_PASSWORD | 123456 | SSH 登录密码 |

- 每次容器启动自动同步凭据，修改 `ports.conf` 后重启即生效
- 支持创建自定义用户（非 root），自动调用 `adduser` 和 `chpasswd`
- 安装和状态检查均显示 SSH 连接信息
