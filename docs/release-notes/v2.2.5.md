# v2.2.5

## 重点修复

- 修复 Magisk 模块同版本重装或降级时用户数据丢失的问题（P0）；改为只要面板数据目录存在即自动备份，不再仅限版本升级场景，并在删除 rootfs 前增加备份完整性校验。
- 修复飞牛 NAS 等第三方 WebView 内打开面板时 API 请求返回 403 的问题；原因是 WebView 发送 `Origin: null`，CORS 中间件未放行该值。

## 新功能

- 用户名支持中文及任意 Unicode 字母，长度放宽至 1-32 位；新增修改用户名接口 `PUT /api/v1/auth/username`，个人资料页可直接修改用户名。
- 订阅管理 Token 鉴权新增「鉴权用户名」字段，支持 Gitee、GitLab 等需要自定义用户名的平台；留空时默认使用 `x-access-token`（GitHub 兼容）。

## 优化

- 备份恢复页面备份内容选择卡片改为 4 列紧凑布局，缩小间距和圆角，整体更简洁。
- 备份文件大小显示改为自适应单位（B / KB / MB / GB），不再固定显示 KB。
- 备份导入增加上传进度条和百分比显示，大文件导入时不再无反馈。
- 定时任务表格左侧「任务名称」和「命令/脚本」列宽缩小，右侧「状态、运行时间、结果、耗时」列宽增大，时间字体从 12px 提升至 13px。
- 全部日志弹窗尺寸整体扩大约 15-25%，包括任务日志、执行日志、订阅拉取日志、脚本执行、面板日志等 9 处。
