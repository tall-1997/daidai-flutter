# v2.2.11

## 修复

- **订阅拉取仍然不建任务（v2.2.10 不彻底）**：在 v2.2.10 的"无 cron 头也兜底建任务"基础上再加 3 重防御：
  1. **通配符白名单识别**：用户在白/黑名单填 `*`、`**`、`*.*`、`全部` 等通配符，视为"不过滤"（旧逻辑字面匹配 → 全部文件被误过滤）
  2. **白名单失效自动放行**：白/黑名单设了但完全没匹配到任何文件 → 自动忽略名单重扫
  3. **filepath.Walk 异常兜底**：某些 NAS / Android 挂载卷下 walk 可能 0 文件返回 → 用 `os.ReadDir` 平铺扫根目录补一遍
