# v2.0.2

## 新功能

### 任务管理

- 新增视图管理功能：支持创建自定义视图，按命令、名称、定时规则、状态、标签、订阅来源等条件筛选任务，方便按订阅分类查看。

### 脚本管理

- 脚本文件侧边栏新增搜索功能，可通过关键字快速定位文件和目录。

### 环境变量

- 变量值旁新增复制按钮，鼠标悬停即显示，支持一键复制变量值。
- 批量改名改为直接输入新变量名模式，选中变量后点击即可统一修改。

### 订阅管理

- 拉取操作新增覆盖选项：订阅级别可设置默认行为（覆盖/保留本地修改），拉取时也可通过下拉菜单临时选择模式。
- 优化 cron 表达式识别：支持 Python docstring（三引号）中声明的 cron 规则，不再要求必须以 `#` 开头。

## 修复内容

### 运行时

- 排除青龙面板专用模块（sendNotify、notify、CryptoJS 等）的自动依赖安装，避免在呆呆面板中误触发安装。
- 修复 Node.js 脚本因 `socks-proxy-agent` 等包缺少 exports 声明导致 `ERR_PACKAGE_PATH_NOT_EXPORTED` 报错的问题，preload 脚本新增兼容性 fallback。
- 检测到 `illegal instruction (core dumped)` 错误时自动输出友好提示，说明 CPU 指令集不兼容的原因及建议。

### 通知推送

- 修复钉钉推送消息不换行的问题，Markdown 格式改为正确的换行语法。
- 优化 Slack 推送标题与正文的段落间距。

### 日志

- 修复部分任务运行后日志显示为空的问题：优化 SSE 流在任务快速完成时的竞态处理，增加前端日志获取的重试次数和间隔。
