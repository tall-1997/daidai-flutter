# v2.2.13

## 新功能

- **定时停止任务**：新增 `定时停止` cron 规则，支持在指定时间自动停止运行中的任务，适合需要在特定时段运行的长驻任务
- **超时永不过期**：任务超时设为 0 时表示永不超时，任务将持续运行直到手动停止或定时停止
- **配置文件 (config.sh)**：环境变量页新增配置文件编辑入口，适合存放不常变动的参数变量，任务执行时自动加载
- **批量停止任务**：任务勾选后新增「批量停止」按钮，可一键停止多个运行中的任务
- **备份任务视图**：备份系统新增「任务视图」选项，恢复后自定义分组视图不再丢失
- **定时备份自动清理**：定时备份完成后自动清理 3 天前的旧备份，避免磁盘空间持续占用

## 优化

- **订阅拉取逻辑重构**：覆盖拉取不再删除本地新增的未跟踪文件，仅覆盖已跟踪文件
- **备份创建对话框**：改为两列布局，增加分类图标，描述文字完整显示不再截断
- **环境变量拖拽**：桌面端新增专属拖拽手柄（三横线图标），避免全行可拖导致操作开关困难
- **视图标签优化**：移除悬停时弹出的编辑/删除按钮，消除布局跳动，管理操作统一收入视图管理弹窗
- **表格复选框增大**：全局表格复选框从 14px 增大到 18px，点击区域扩大，触控更友好
- **依赖管理筛选**：新增「已安装/失败」快速筛选标签页，直观显示各状态数量

## 修复

- **仪表盘表格对齐**：修复最近执行任务表头与数据列不对齐的 CSS 优先级问题
- **侧边栏图标居中**：修复菜单收起后图标未正确居中的问题
- **脚本版本对比**：修复历史版本对比界面 Monaco 编辑器仅显示几像素高度的问题
- **面板日志格式**：日志输出改为 `时间 [级别] [IP] 方法 路径 状态=码` 格式，更清晰易读
- **面板日志滚动**：打开面板日志后自动滚动到底部，刷新时保持在最新位置
- **Magisk 依赖持久化**：重启后自动恢复并重装丢失的 Node.js/Python 依赖，新增 deps 目录持久化快照机制
